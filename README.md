# task-app
# https://roadmap.sh/projects/task-tracker
A task app for the CLI with python. 

The cli app is used from the terminal in your IDE or pc terminal
to add a task, in your terminal write the following
python3 project_name add

to update a task
python3 project name update

to list all tasks
python3 project_name list-tasks

to list done tasks
python3 project_name list done

to list in-progress tasks
python3 project_name list progress

to list to-do tasks
python3 project_name list to-do

to chnage the progress of each task
python3 project_name progress
